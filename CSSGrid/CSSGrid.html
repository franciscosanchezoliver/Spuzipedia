CSS Grid

  • Grid layout system with row and columns.
  • No need to use floats and positioning.

Un layout Grid tiene que tener un padre con el atributo "display" a "grid" o "inline-grid" 

<div class = "grid-container">
	<div class= "grid-item">1</div> 
	<div class= "grid-item">2</div> 
	<div class= "grid-item">3</div> 
	<div class= "grid-item">4</div> 
	<div class= "grid-item">5</div> 
</div>			
		
.grid-container {
	display: grid;
	grid-template-columns: auto auto auto;
	background-color: #2196F3;
	padding: 10px;
	}

Resultado:
1
2
3
4
5

CSS Grid

  • Grid layout system with row and columns.
  • No need to use floats and positioning.

Un layout Grid tiene que tener un padre con el atributo "display" a "grid" o "inline-grid"

Display "grid"

<div class = "grid-container">
	<div class= "grid-item">1</div> 
	<div class= "grid-item">2</div> 
	<div class= "grid-item">3</div> 
	<div class= "grid-item">4</div> 
	<div class= "grid-item">5</div> 
</div>			
.grid-container {
	display: grid;
	grid-template-columns: auto auto auto;
}

Gap en un Grid

Los gaps son espacios en blanco en el grid, hay 2 tipos:

  • grid-row-gap: espacio entre filas.
  • grid-column-gap: espacio entre columnas.

El "gap" es un espacio que podemos poner para separar entre cajas del grid. En el siguiente ejemplo podemos ver un ejemplo sencillo.

grid-column-gap

.contenedor-con-gap{
	display:grid;
	grid-column-gap: 50px;
}

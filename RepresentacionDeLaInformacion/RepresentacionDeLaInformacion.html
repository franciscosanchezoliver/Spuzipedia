Representación y comunicación de la información: elementos constitutivos de un sistema de información

  • El bit es la unidad básica de información. Puede tener 2 valores (0 o 1).
  • 1 byte = 8 bits (octeto)
  • El bit se representa con una b minúscula y el byte con una mayúscula. 8b = 1B. Por tanto un B es un octeto (8 bits), y por tanto 8Gbits/s equivale a 1GByte/s
  • 1 nibble = 4 bits (cuarteto). Se utiliza para el sistema hexadecimal (de 0 a F => 16 valores = 2^4).

Sistemas numéricos

  • Binario: base 2. Utiliza el 0 y el 1.
  • Octal: base 8. Del 0 al 7.
  • Decimal: base 10. Del 0 al 9.
  • Hexadecimal: base 16. Del 0 al 9 y de la A a la F.

Conversión de decimal a otros sistemas

Decimal a binario

Dividimos el número entre 2 sucesivamente. Los restos ordenados a la inversa forman el resultado.

Ej: 810 => X2

  • 8/2 = 4 ; resto=0
  • 4/2 = 2 ; resto=0
  • 2/2 = 1 ; resto=0
  • 1/2 = 0 ; resto=1

810 => 10002

Decimal a octal

Dividimos el número entre 8 sucesivamente. Los restos ordenados a la inversa forman el resultado.

Ej: 3410 => X8

  • 34/8 = 4 ; resto = 2
  • 4/8 = 0 ; resto = 4

3410 => 428

Decimal a hexadecimal

Primero convertimos el número a binario, y de binario a hexadecimal.

Ej: 18310 => X16

  • 18310 => X 2
  • X2 => Y16
Por tanto
  • 183/2 = 91 ; resto = 1
  • 91/2 = 45 ; resto = 1
  • 45/2 = 22 ; resto = 1
  • 22/2 = 11 ; resto = 0
  • 11/2 = 5 ; resto = 1
  • 5/2 = 2 ; resto = 1
  • 2/2 = 1 ; resto = 0
  • 1/2 = 0 ; resto = 1
18310 => 101101112

Conversión de otros sistemas numéricos a decimal

Conversión de binario a decimal

Para convertir de binario a decimal cada dígito es multiplicado por la base 2 elevada a la posición que ocupa.

Ej: 1012 => X10

  • 1 · 22 + 0 · 21 + 1 · 20 = 4 + 0 + 1 = 5

Ej 2: 110112 => X10

  • 1·24 + 1·23 + 0·22 + 1·21 + 1·20 = 16 + 8 + 0 + 2 + 1 = 27

Conversión de octal a decimal

Para convertir de octal a decimal cada dígito es multiplicado por la base 8 elevada a la posición que ocupa.

Ej: 3568 => X10

  • 3·82 + 5·81 + 6·80 = 192 + 40 + 6 = 238

Conversión de hexadecimal a decimal

Cada digito es multiplicado por la base 16 elevada a la posición.

Ej: A0D16 => X10

  • A(10)·162 + 0·161 + D(13)·160 = A·256 + 0 + 13·1 = 2560 + 0 + 13 = 2573

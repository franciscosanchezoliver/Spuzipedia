Java 8. Lambda Expressions

  • El mayor cambio introducido en Java 8.
  • Facilitan la programación funcional.
  • Simplifican el desarrollo.

Sintaxis

Una expresión lambda se caracteriza por tener el siguiente formato:

 parametros -> cuerpo de la expresión
    Las características de las expresiones lambdas son:
  • Declaración opcional de los parámetros. Si los parámetros no son declarados entonces el compilador los inferirá según el valor que contenga la variable.

Ejemplo de Parámetros opcionales

		
 public class LambdaExample {
 
 	interface OperacionMatematica {
 		int calcular(int a, int b);
 	}
 
 	public static int operar(int a, int b, OperacionMatematica operacionMatematica) { 
 		return operacionMatematica.calcular(a, b);
 	}
 
 	public static void main(String args[]) {
 		// Tipo de parámetros sin declarar, inferidos por el compilador
		OperacionMatematica resta = (a, b) -> a - b;
		 
 		// Tipo de parámetros declarados
		OperacionMatematica suma = (int a, int b) -> a + b;
		 
 		System.out.println(operar(1, 2, suma));
 		System.out.println(operar(1, 2, resta));
 	}
 
 }

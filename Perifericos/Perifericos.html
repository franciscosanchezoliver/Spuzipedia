Perifericos

  • Puertos: interfaz entre un ordenador y otros ordenadores o periféricos.
  • Periférico: dispositivo que se conecta al ordenador a través de un puerto y hace algo.
  • 2 tipos
    • Entrada: teclado, ratón, escáner, lector de huellas, etc.
    • Salida: pantalla, altavoz, impresora.
  • Tipos de USB
    Versión 1.0 2.0 3.0 (SS, Super Speed) 3.1
    Velocidad 1.5-12Mbits/s 480Mbits/s 5Gbits/s 10Gbits/s
    Características Conector azul
  • Tipos de USB por forma:
    • A: es el USB estándar plano.
    • B: es el USB cuadrado.
    • A y B: tienen 4 pines de voltaje (Voltaje, Data-, Data+ y tierra).
  • Otros puertos:
    • eSata: External Sata.
    • Ethernet.
    • Firewire: diseño por apple.
    • HDMI/DVI/VGA.
    • Jack
    • SCSI: para discos duros y cintas. (Small Computer System Interface).
    • PS/2: Conector de 6 pines para teclado y ratón. Aunque electrónicamente son similares, pueden no ser intercambiables (verde para el ratón y violeta para teclado).
    • Puerto paralelo: envia varios bits a la vez.

Perifericos

  • Puertos: interfaz entre un ordenador y otros ordenadores o periféricos.
  • Periférico: dispositivo que se conecta al ordenador a través de un puerto y hace algo.
  • 2 tipos
    • Entrada: teclado, ratón, escáner, lector de huellas, etc.
    • Salida: pantalla, altavoz, impresora.

USB por forma

  • A: es el USB estándar.
  • B: es el USB cuadrado.
  • A y B: tienen 4 pines de voltaje (Voltaje, Data-, Data+ y tierra).
    Pin Señal Color Descripción
    1 Vcc +5V
    2 D- Data-
    3 D+ Data+
    4 GND Masa

Otros puertos

  • eSata: External Satades de discos duros externos.
  • Ethernet.
  • Firewire: diseñado por apple.
  • Para pantallas:
    • HDMI.
    • DVI: para pantallas LCD.
    • VGA:
  • Jack (conector Audio Analógico): para conectar micrófonos, auriculares, y otros sistemas de señal analógica.
  • SCSI: para discos duros y cintas. (Small Computer System Interface).
  • PS/2: Conector para teclado y ratón con 6 pines . Aunque electrónicamente son similares, pueden no ser intercambiables (verde para el ratón y violeta para teclado).
  • Puerto paralelo
    • Envia varios bits a la vez -> necesita varias líneas de datos.
    • Utlizado para las impresoras hasta que llego el USB.
  • Puerto serie: transmite un bit a la vez.
  • Thunderbolt
    • Creado por Apple.
    • Thunderbolt 1 : 10 Gbit/s
    • Thunderbolt 2 : 20 Gbit/s
    • Thunderbolt 3 : 40 Gbit/s

Tipos de teclado

  • QWERTY: distribución más común.
  • QWERTZ/QWERTU: regiones germanohablantes.
  • AZERTY: para francófonos
  • HCESAR: usando antes en portugal (ahora se usa el QWERTY).
  • Dvorak: es más ergonómico para reducir los errores ortográficos.

Elementos de impresión

La impresoras se suelen conectar con un puerto paralelo, o un USB. Si es una impresora en red entonces está conectada por un RJ-45.

Según cómo escriban

  • 2 tipos de impresoras
    • Impacto
      • Golpean el papel y el impacto transfiere la tinta.
      • Para texto o gráficos muy sencillos.
      • Monocromáticas
      • Según el cabezal de impresión, hay 2 tipos
        • Margarita
          • Tiene una bola metálica con relieves de letras y símbolos.
        • De agujas (Matriciales)
          • Tiene una matriz de agujas que impactan sobre el papel.
          • Cuantas más agujas tenga mayor será la resolución.
    • Sin impacto
      • Térmicas: agujas calientes tocan papel termo sensible (cajeros).
      • Inyección o chorro de tinta: tira gotas de tinta.
      • Sublimación de tinta: se usa calor para transferir la tinta.
      • Laser.

Según el método de impresión

    Hay 3 tipos:
  • Carácter a carácter
    • Matriciales.
    • Margarita.
    • Térmica.
    • Inyección.
  • Línea a línea
  • Cinta.
  • Cadena.
  • Tambor.
  • Página a página: laser

Lenguajes de descripción de página (LDP)

  • Describen el formato de la página (colación del texto, gráficos, etc.).
  • Hay 3 tipos
    • PostScript: 1982 por Adobe Systems.
    • PCL (Printer Command Language)
      • Desarrollado por HP.
      • Para impresoras de chorro de tinta.
      • Actualmente está la versión PCL6
    • HPGL: para plóter (impresora lineal para ambitos científicos.)

Elementos de almacenamiento

Interfaces

  • Parallel ATA (PATA)
    • Interfaz para conectar dispositivos de almacenamiento como discos duros, floppy disk y lectoras de CD.
    • La extensión ATAPI permite conectar con lectoras de CD.
  • Serial ATA (SATA)
    • Reemplata a PATA.
    • Sirve para discos duros, lectoras de CD, y discos de estado solido SSD.
    • Version Velocidad (el doble) Nombre
      SATA 1.0 1.5 Gbit/s = 150 MB/s SATA-150
      SATA 2.0 3 Gbit/s = 300 MB/s SATA-300
      SATA 3.0 6 Gbit/s = 600 MB/s SATA-600

RAID

  • Redudant Array of Independent Disks.
  • Sistema de almacenamiento de datos.
  • Varios discos duros o SSD.
  • Para distribuir o replicar los datos.
  • Beneficios
    • + capacidad de la BD.
    • + tolerancia a fallos
    • + integridad.
    • + tasa de transferencia.

Niveles de RAID

RAID 0

  • Distribuye los datos equitativamente entre 2 o más discos.
  • Sin información de paridad, por tanto no hay redundancia.
  • No hay tolerancia a fallos.
  • Fallaría todo el conjunto si falla uno de los discos. Por esta razón cuantos más disco tenga el conjunto mayor será la posibilidad de fallo (la fiabilidad del sistema baja cuantos más discos haya).
  • Alto rendimiento en la escritura ya que se puede escribir en varios discos a la vez de forma paralela.
  • Bajo rendimiento lectura, ya que los sectores que se buscan pueden encontrarse en diferentes discos.
  • El espacio de almacenamiento que se añada está limitado por el disco más pequeño. Por ejemplo si tenemos un disco de 400 GB y otro disco de 150 GB entonces el tamaño total del RAID 0 será de 300 GB.

RAID 1 (espejo)

  • Copia exacta (espejo) en uno o más discos.
  • Aumenta la seguridad pero usamos más almacenamiento. Si por ejemplo se rompe un disco, tendríamos una copia del mismo.
  • Un conjunto RAID 1 está limitado por el más pequeño de sus discos.
  • Aumenta mucho la fiabilidad ya que para que falle el conjunto tendrían que fallan todos los discos.
  • Aumenta la velocidad de lectura ya que si es necesario leer un dato estará disponible en varios discos duros de donde se puede sacar dicho dato.
  • El rendimiento de escritura no mejora, ya que cuando hay que escribir un dato, este se tiene que escribir en todos los discos. Por tanto el tiempo de escritura es el mismo para 1 disco que para varios.

RAID 2

  • Reparto de la información a nivel de bits (y no a nivel de bloques como los anteriores).
  • Hay paridad con el código de Hamming .
  • No hay redundancia.

RAID 3

  • Reparte los datos a nivel de bytes (en lugar de a nivel de bloques).
  • Se utiliza un disco de paridad. Por tanto necesita un mínimo de 3 discos.
  • Tasa de transferencia muy altas.

RAID 4 (IDA)

  • Divide a nivel de bloque.
  • Usa un disco de paridad. Por tanto necesita un mínimo de 3 discos.
  • Rendimiento de lectura alto ya que se pueden hacer varias peticiones de lectura a la vez.
  • Rendimiento de escritura lento ya que cada vez que hay que escribir hay que mantener el disco de paridad, y solo hay un disco de paridad.

RAID 5

  • Se dividen los datos por bloques.
  • La información de paridad se distribuye entre todos los discos duros.
  • Si falla un disco se puede reconstruir la información gracias a la información de paridad.

RAID 6

Como el RAID 5 pero con 2 bloques de paridad.

Elementos de visualización

    • Monitores LCD (Liquid Cristal Display).

Elementos de digitalización

Hay varios tipos de escáneres:
  • De mano: se mueven a mano por lo que se quiera escanear.
    Hay 2 tipos:
    • De documentos.
    • 3D.
  • De tambor:
    • Muy grandes.
    • Permiten digitalizaciones de alta calidad (hasta 4000ppp)
  • De cama plana
    • Son los más comunes.
    • Se pone el papel sobre el cristal.
  • Cenital:
    • Se hace una fotografía desde arriba.
    • Se usa para no dañar el papel que se está digitalizando (cómo libros antiguos).

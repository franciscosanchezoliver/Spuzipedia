Forensics Warmup 2

Hmm for some reason I can't open this PNG? Any ideas?

  • Get the file: wget url
  • If we use the file command we'll realise that the file even that the name is "flag.png" its extension is jpeg. And this happens because in linux the extension of a file doesn't have to match the name.
  • We have to rename the file from flag.png to flag.jpg: mv flag.png flag.jpg

Solution: picoCTF{extensions_are_a_lie}

General Warmup 1

If I told you your grade was 0x41 in hexadecimal, what would it be in ASCII?

ASCII: character encoding for electronic communication. ASCII codes represents text in computers.

As we are interest in the vlaue 0x41, this is the following row in the table:

  • Hx: 41
  • Dec: 65
  • Oct: 101
  • Chr: A

Python script 

	#!/usr/bin/evn python

	print "picoCTF{%s}" % chr(0x41)

Solution: picoCTF{A}

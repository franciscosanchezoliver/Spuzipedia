General Warmup 1

If I told you your grade was 0x41 in hexadecimal, what would it be in ASCII?

ASCII: character encoding for electronic communication. ASCII codes represents text in computers.

As we are interest in the vlaue 0x41, this is the following row in the table:

  • Hx: 41
  • Dec: 65
  • Oct: 101
  • Chr: A

Python script 

	#!/usr/bin/evn python

	print "picoCTF{%s}" % chr(0x41)

Solution: picoCTF{A}

General Warmup 2

Can you convert the number 27 (base 10) to binary (base 2)?

27 |2
    ---
1    13 | 2
         ---
      1   6 | 2
             ---
          0   3 | 2
                 ---
              1   1

We have to get the number backwards, so 11011.

Pyhon script: 

#!/usr/bin/env python

# ****************************
# * Convert number to binary *
# ****************************

# this will get the following format result:
# bin(27) -> 0b11011
binario =  bin(27)

# remove the  first two characters
# 0b11011 -> 11011
binario = binario[2:]

# print the result of the picoCTF challenge
print ("picoCTF{" + binario + "}")

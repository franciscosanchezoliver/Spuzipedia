React

  • Para usar react necesitamos tener instalado node. Podemos comprobar si lo tenemos instalado usando el comando: node --version

    Si no lo tenemos instalados vamos a la página de node: https://nodejs.org/es/

  • Una vez instalado node. Vamos a la página de React (en la parte dónde te dice como crear una nueva app en React): https://reactjs.org/docs/create-a-new-react-app.html

    Instalamos una herramienta de React que nos ayudará a crear la arquitectura de una app en React básica. Para ello ejecutamos el comando: npm install -g create-react-app

  • Creamos una carpeta donde queramos alojar nuestro proyecto. Abrimos una consola de comandos, y ejecutamos el comando create-react-app myfirstapp

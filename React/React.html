React

  • Para usar react necesitamos tener instalado node. Podemos comprobar si lo tenemos instalado usando el comando: node --version

    Si no lo tenemos instalados vamos a la página de node: https://nodejs.org/es/

  • Una vez instalado node. Vamos a la página de React (en la parte dónde te dice como crear una nueva app en React): https://reactjs.org/docs/create-a-new-react-app.html

    Instalamos una herramienta de React que nos ayudará a crear la arquitectura de una app en React básica. Para ello ejecutamos el comando: npm install -g create-react-app

  • Creamos una carpeta donde queramos alojar nuestro proyecto. Abrimos una consola de comandos, y ejecutamos el comando create-react-app myfirstapp

  • Arrancamos el proyecto creado con el comando: npm start

Componentes

La idea en React es que todo sean componentes, que son pequeños trozos de código. Un ejemplo de componente sería: 

import React , {Component} from 'react';

class Navegacion extends Component{
	render(){
		return (
			<div>
				Ejemplo de componente
			</div>
		)
	}
}

export default Navegacion;

Y para llamarlo desde la Aplicacion principal tendriamos que hacer importar este nuevo componente con: import Navegacion from './components/Navegacion';

Y pintar este componente llamandolo así: <Navegacion/>

Parar un parametro a un componentes

Al llamar al componente podemos pasarle variables <Navegacion titulo="Mi primer componente"/>

Dentro del código del componente accederiamos a dicha variable asi: {this.props.titulo}

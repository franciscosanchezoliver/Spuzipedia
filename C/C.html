Hello World

Los comandos "include" son comandos preprocesador. Cuando creamos un proyecto vacio y vemos estos includes ya estaríamos incluyendo código para que pueda funcionar minimamente. stdio.h esta librería se encarga de poder imprimir por pantalla. La librería stlib.h permite al compilador detectar la función main, el main será el punto de entrada de la aplicación, por tanto todo proyecto tendrá una función main. 

 #include <stdio.h>
 #include <stdlib.h>

 // función que no tiene parámetros de entrada
 // función devuelve un int
 // el valor de retorno 0 indica que el programa 
 // se ha ejecutado correctamente.
 int main(){
	// la función printf se encuentra en la librería stdio.h
	printf("Hello world!\n"); 
	return 0;
 }

scanf

Nos permite leer diferentes tipos de datos, como por ejemplo enteros, decimales, o cadenas de caracteres.

scanf se encuentra en la librería "stdio.h".

En la variable de tipo puntero "format" le pasamos el tipo de datos que queremos leer. Si todo es correcto devuelve el número de datos leidos, si hay algún error devuelve EOF. 

#include <stdio.h>
int scanf(const char *format, ...);

Si usamos el especificador de formato %s es para leer cadenas. La función scanf leerá caracteres hasta encontrar un espacio, un intro, un tabulador, un tabulador vertical o un retorno de carro. Al final de lo que ha leído añade un caracter nulo automáticamente.

NOTA: Los caracteres leídos se guardan en un array, por lo que el array debe ser lo suficientemente grande para poder guardar lo leído. Por esto es peligroso usar scanf o gets para leer cadenas, ya que scanf admite que el usuario escriba más caracteres que lo que el array definido puede admitir. Como resultado scanf escribirá los caracteres que ya no quepan en el array definido en otras porciones de memoria que pueden contener otros datos que está usando nuestro programa.

A scanf hay que pasarle punteros a los argumentos. En el siguiente ejemplo le pasamos un puntero a la variable: 

int entero1, entero2;
printf("Introduce dos enteros separados por un espacio: \n");
scanf("%d %d", &entero1, &entero2);

Sin embargo cuando leemos una cadena de caracteres, los caracteres leídos (más el caracter nulo de terminación) se guardan en el array apuntado. Aquí no se usa el operación de dirección "&" porque el propio nombre de un array (en este caso el array contiene una cadena) apunta a la dirección de comienzo del propio array. 

char cadena[8];
scanf("%s" , cadena);

Ejemplo de programa con scanf: 

#include <stdio.h>
#include <stdlib.h>

#define TAMANIO_MAXIMO 80

int main()
{
	char cadena[TAMANIO_MAXIMO];
	int entero1, entero2;
	float decimal;

	// Lectura de Enteros
	// A scanf hay que pasarle punteros a los argumentos. En el siguiente ejemplo
	// le pasamos un puntero a la variable:
	printf("Introduce dos enteros separados por un (espacio, tabulador, intro ): \n");
	scanf("%d %d", &entero1, &entero2);

	// Lectura de decimales
	printf("Introduce un número decimal:\n");
	scanf("%f" , &decimal);

	// Lectura de cadenas
	// Cuando leemos una cadena de caracteres, los caracteres leídos
	// (más el caracter nulo de terminación) se guardan en el array
	// apuntado. Aquí no se usa el operación de dirección "&" porque
	// el propio nombre de un array (en este caso el array que contiene
	// una cadena) apunta a la dirección de comienzo del propio
	// array.
	scanf("%s", cadena);

	printf("Esto es todo lo que has escrito:\n");
	printf("%d %d %f %s\n", &entero1, &entero2, decimal,cadena);

	return 0;
}

atoi

  • Convierte una cadena en un entero.
  • Si la conversión no se pudo hacer devuelve 0.
  • int atoi (const char *str)

Un ejemplo de programa sería: 

#include <stdio.h>
#include <stdlib.h>
#include <string.h>

int main()
{
	int unEntero;
	char unString[20];

	//La funcion atoi se utiliza para convertir un string a un entero.

	// Guardamos la cadena en la variable str
	strcpy(unString, "1563");
	//Convertimos el valor de string a entero
	unEntero = atoi(unString);
	printf("Conversion de la cadena %s a entero %d \n", unString, unEntero);

	// Si la conversion no se puede hacer entonces devolvera 0
	strcpy(unString, "hola mundo");
	unEntero = atoi(unString);
	printf("Si la conversion no se puede hacer debe devolver 0. \n");
	printf("Si convertimos 'hola mundo' a entero nos da %d" , unEntero);

	return 0;
}

Paso de parámetros a funciones

Podemos llamar a una funcion y pasarle los parámetros por valor o por referencia. 

	
#include <stdio.h>

int main()
{
	// **** P O R   V A L O R ****

	// Paso de parametro por "VALOR".
	// Estoy pasando una el valor que contiene la variable "dinero"
	void ganarPorValor(int dinero_ganado)
	{
		printf("El valor del dinero es: %d \n", dinero_ganado);
	}

	int dinero = 500;
	// Pasamos el parametro por valor, es decir pasamos el valor 500
	ganarPorValor(dinero);

	// **** P O R    R E F E R E N C I A ****

	// Imaginemos que la siguiente tabla es la memoria RAM
	// Y cada celda es un fragmento de la memoria en la que se
	// puede guardar informacion. Cada celda ademas esta numerada
	// para poder acceder a dicha celda:

	//      0         100        200        300
	//  ------------------------------------------
	// |         ||         ||         ||         |
	//  ------------------------------------------

	// Supongamos que la variable dinero se encuentra en la posicion 200
	// de la memoria, y que tiene el valor de 17:

	//      0         100        200        300
	//  --------------------------------------------
	// |         ||         || dinero=17 ||         |
	//  --------------------------------------------

	// Cuando hacemos el paso por referencia, usamos el caracter '&'
	// Y esto significa una UBICACION DE LA MEMORIA RAM.
	// Con esto estamos indicando que vamos a acceder a la posicion de
	// memoria 200 y tenemos TODO el control de todo lo que hay ahi.

	// Para poder recibir una referencia e interpretarla necesitamos
	// usar lo que se llaman "punteros". Esta tipo de variable "puntero"
	// apunta a una zona de memoria.
	// Para usar punteros utilizamos el caracter '*'

	// Ahora las variables "money" y "dinero_ganado" son una sola variable
	// con diferentes nombres ya que apuntan a la misma zona de memoria.
	// Por lo tanto en el momento que hemos usado punteros, las variables ya
	// estan unidas y si se modifica una se modificara tambien la otra.
	void ganarPorReferencia(int * dinero_ganado)
	{
		printf("El valor del dinero es %d \n", *dinero_ganado);

		// Como hemos usado un puntero a la zona de memoria de "money"
		// si modificamos el valor, estaremos modificando la zona de memoria
		// y por tanto cuando salga de este método la variable "money" de main
		// tambien se modificara
		* dinero_ganado = 23;
	}
	int money = 500;
	ganarPorReferencia(&money); // Paso por referencia

	printf("El valor money en MAIN es %d \n", money);

}

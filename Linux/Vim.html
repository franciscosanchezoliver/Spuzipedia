Vim

Modos

  • Normal
    • Para navegar por el texto.
    • Para salir pulsamos "ESC".
    • Ayuda-> ":help Normal-mode"
  • Insertar
    • Para insertar nuevo texto.
    • Ayuda-> ":help Insert-mode"
  • Visual
    • Para manipular trozos de texto seleccionado. Podremos ejecutar comandos con el texto seleccionado.
    • Ayuda-> ":help Visual-mode"
  • Select
    • Muy parecido al modo "Visual" pero con funciones parecidas a Microsoft.
  • Command Line
  • Para meter comandos.
  • Ayuda-> ":help Command-line-mode".
  • Ex-Mode
    • Como el modo de comando pero optimizado para el procesador batch.
    • Ayuda-> ":help Ex-mode"

Para salir de un modo tenemos que presionar "ESC".

Insertar y Reemplazar

Insertar lo utilizamos para escribir texto. Podemos a entrar a este modo de las siguientes maneras:

  • a: append after cursor.
  • A: append at end of line.
  • i: insert before cursor.
  • I: insert at beginning of line.
  • insertar texto: le damos a la "i" para meternos en el modo insertar.
  • borrar línea: para borar una linea pulsamos "dd"

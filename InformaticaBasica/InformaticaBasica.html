Representación y comunicación de la información: elementos constitutivos de un sistema de información

  • El bit es la unidad básica de información. Puede tener 2 valores (0 o 1).
  • 1 byte = 8 bits (octeto)
  • El bit se representa con una b minúscula y el byte con una mayúscula. 8b = 1B
  • 1 nibble = 4 bits (cuarteto). Se utiliza para el sistema hexadecimal (de 0 a F => 16 valores = 2^4).
  • Bit (b) < Byte (B) < Kilobyte (kB) < Megabyte (MB) < Gigabyte (GB) < Terabyte (TB) < Petabyte (PB) < Exabyte (EX) < Zettabyte (ZB) < Yottabyte (YB)
    Nombre Símbolo Valor
    bite b 0 o 1
    Byte B 8 bits

Sistemas de altas prestaciones. Grid computing.

Definición

Un sistema de altas pretaciones es un ordenador con grandes capacidades de cómputo, capaz de procesar grandes volúmenes de información. Puede ser tanto en el ámbito de la gestión como del ámbito técnico-científico.
Podemos diseñar un sistema de altas prestaciones de dos formas:

  • Usando varios procesadores (Multiprocesadores): Usando una sola máquina pero con mucha potencia de cálculo, para ello es necesario utilizar la fuerza de varios procesadores.
  • Usando varias máquinas (Multicomputadores): se usan varios ordenadores para diseñar el sistema. El sistema tendrá la suma de las potencias de las máquinas.
Podemos hacer otras divisiones dentro de cada tipo:
  • Multiprocesadores
    • SMP(Symmetric Multi-Processing)
    • MPP(Massively parallel processors)
  • Multicomputador
    • Grid
    • Clúster
Para los multiprocesadores usaremos la taxonomía de Flynn. Para los multicomputadores se harán de 2 formas: Grid y Clúster.

Multiprocesadores. Taxonomía de Flynn

Es una clasificación de ordenadores según su arquitectura. Esta clasificación se basa en el flujo que siguen los datos dentro de la máquina y de las instrucciones que se han hecho sobre estos datos.
Podemos definir:

  • Flujo de instrucciones: conjunto de instrucciones secuenciales que se enjecutan en un único procesador.
  • Flujo de datos: flujo de datos requeridos por el flujo de instrucciones.
Flynn hace 4 clasificaciones dentro de los multiprocesadores:
  • SISD (Single Instruction Single Data)
  • SIMD (Single Instruction Multiple Data)
  • MISD (Multiple Instruction Single Data)
  • MIMD (Multiple Instruction Multiple Data)

SISD

Como indica el nombre (Single Instruction Single Data) son ordenadores que su arquitectura solamente permite ejecutar una única instrucción con un solo dato en cada ciclo de reloj. Esta sería la arquitectura de Von-Neumann. Esta arquitectura no usa el paralelismo ni en las instrucciones ni en los datos, aunque se puede mejorar el rendimiento usando ténicas de segmentación o pipelining.
La memoría del ordenador se divide en dos partes:

  • Parte para instrucciones: las instrucciones que se van a ejectuar, esto es el programa.
  • Parte para datos: los datos sobre los que se van a ejecutar las instrucciones.
Es necesario una Unidad Central de Procesamiento(CPU) que es el módulo que hace las operaciones matemáticas necesarias para poder ejectuar las instrucciones del programa. Una CPU tiene que tener como mínimo la capacidad de sumar y de realizar operaciones lógicas.

Sistemas de ficheros

    Controlan cómo se almacenan/recuperan los datos en los distintos medios.
  • FAT (File Allocation System):
    • 3 variantes: FAT 12, 16, 32
    • Utilizado cuando NTFS no es apropiado.
  • NTFS (New Technology File System): para windows 7,8, y 10
  • EXT:
    • Para el Núcleo de Linux.
    • Variantes: EXT, EXT2, EXT3, EXT4
    • A partir de EXT3 tiene la tecnología "journaling" que protege los archivos si hay un apagado abrupto del sistema.
  • JFS
    • Creado por IBM
    • Tiene journaling.
  • XFS: creado por Silicon Graphics.
  • ZFS: creado por Oracle.
  • ReiserFS: creado por Namesys.
  • HFS Plus: creado por Apple.
  • ISO 9660: sistema de archivos para medios ópticos.
